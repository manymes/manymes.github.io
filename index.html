How-To: Install manymes-chrome-extension

  1. unzip manymes-chrome-extension.zip
  2. open your terminal
  3. cd into manymes-chrome-extension folder and run bower install
  4. open chrome://extensions/
  5. activate developer mode (checkbox)
  6. press load unpacked extensions...
  7. choose app folder in the unpacked manymes-crhome-extension folder
  8. activate browsing by clicking on the meaningfull icon in your toolbar
  9. enjoy.
